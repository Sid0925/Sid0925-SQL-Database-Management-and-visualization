SQL Database Management and visualization


Project Objective:

Developed a data pipeline using SQL and Power BI to extract data from multiple sources, transform it, and load it into a data model that could be easily queried and visualized. This project resulted in a more efficient process for accessing and analysing data, and enabled more timely and accurate reporting and decision-making. Project involved using SQL and Power BI to extract, clean, and analyse data from a database, and create interactive visualizations to communicate findings and inform decision-making. The project resulted in improved understanding of trends and patterns in the data, and enabled more informed decision-making based on data-driven insights
