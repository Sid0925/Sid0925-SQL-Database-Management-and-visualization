Project Data Pipeline






Project Structure


1) I got multiple CSV files as a every each category of data which has been continiousely stored by the client from the cake shop.
2) I had to upload that each CSV file on the SQL SERVER DATABASE in order to create A Data Pipeline.
3) Then, I established the persies Relationship between each tables, basically each CSV file which has been given to me became the tables on SQL server database.
4) Furthermore, Thoes tables have been derived through power BI for visualization.
5) In PowerBI, I made different kind of graphs to demonstrate the sales and relationship factors for sales. Also made different Measures and calculated columns(DAX formula in powerbi) for specific business needs.
6) I have provided, A Time series analysis dashboard where we have predicted further 10 days prediction sales.

Objective

• Extracting and analyzing data from a database using SQL queries, and visualizing the results using Power BI
• Implementing a business intelligence solution that allows users to explore and analyze data using interactive dashboards and reports created with Power BI
• Creating a data pipeline that ingests data from multiple sources, transforms it using SQL, and loads it into a data model that can be queried and visualized using Power BI
• Building a data analytics platform that integrates SQL and Power BI to enable users to gain insights from data stored in a database

Achivements
Designed and implement a data pipeline to efficiently store and manage large amounts of data from multiple CSV files in a SQL Server database.
use Power BI to create visualizations and insights from the data stored in the SQL Server database, enabling stakeholders to easily understand and make decisions based on the data.
improved data accessibility and analysis for the organization by creating a central repository for data from multiple sources and making it available for visualization and reporting through Power BI.
Demonstrated expertise in database design, data integration, and data visualization techniques, and the ability to apply these skills to solve business problems.
